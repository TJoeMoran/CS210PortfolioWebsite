Battleship Game (CS-150 Fall 2024)

This is my Battleship game from CS-150. I completed this as a group project with 3 classmates. It's coded entirely in C++. The game allows the user to play Battleship against the computer. It lets the user place their ships manually or randomly. It tracks and displays hits, misses, and sunk ships and reports the winner at the end. It also reports the result of each move to a log file.

Command Line Steganography Tool (CS-151 Spring 2025)

This is my stegoBMP command line steganography tool for CS-151. Steganography is the process of concealing a message or other file within a file. This tool works with 24-bit uncompressed BMP image files to either encode or decode messages. It utilizes the unused bits in the binary code of the image to store a message, and it can also decode from an image that contains a hidden message.

Assembly Arithmetic Calculator (CS-155 Spring 2025)

This is my assembly language calculator that I wrote for my CS-155 final project. It's a basic calculator that performs addition, subtraction, multiplication, and integer division on two-digit values (00 - 99). It performs input validation, checks for divide by 0 errors, and loops until user chooses to quit the program.

Stock Data Web Scraper (CS-270 Fall 2025)

This is my Python web scraper that I coded for CS-270. It uses BeautifulSoup to scrape Yahoo Finance for stock data, and it extracts and stores each stock price. Each time the user runs a new check, the program reports the current prices and how much each price changed since the last check.

Lexical Analyzer (CS-210 Fall 2025)

This is my lexical analyzer, or lexer, that I coded for my CS-210 midterm project. It breaks the input into lexemes or tokens and labels them by type (comment, keyword, operator, etc.). This lexer is written in C and was tested using programs written in CCX, an imaginary language with syntax similar to C and C++.
